
호출 방식 정리


1. 학식메이트 매칭 성공 시

chatService.createMealChatRoom(
        memberAId,
        memberBId
);

예시:

Long memberAId = matching.getRequester().getMemberId();
Long memberBId = matching.getReceiver().getMemberId();

chatService.createMealChatRoom(memberAId, memberBId);

생성 결과:

room_type = DIRECT
matching_mode = MEAL
두 명 모두 MEMBER


2. 소개팅 매칭 성공 시
chatService.createBlindDateChatRoom(
        memberAId,
        memberBId
);

예시:

Long memberAId = matching.getRequester().getMemberId();
Long memberBId = matching.getReceiver().getMemberId();

chatService.createBlindDateChatRoom(memberAId, memberBId);

생성 결과:

room_type = DIRECT
matching_mode = BLIND_DATE
두 명 모두 MEMBER


3. 과팅 매칭 성공 시

과팅은 대표자 2명과 일반 참여자 리스트를 나눠서 넣기.
과팅 채팅방 생성 시 leaderMemberIds에는 각 팀 대표자 2명만 넣고,
memberIds에는 대표자를 제외한 일반 참여자만 넣어야 하며,
같은 회원 ID가 두 목록에 중복으로 들어가면 안 됨.

List<Long> leaderMemberIds = List.of(teamALeaderId, teamBLeaderId);

List<Long> memberIds = matchedMemberIds;

chatService.createGroupDateChatRoom(
        "과팅 채팅방",
        leaderMemberIds,
        memberIds
);

예시 2대2:

List<Long> leaderMemberIds = List.of(
        teamALeaderId,
        teamBLeaderId
);

List<Long> memberIds = List.of(
        teamAMember1Id,
        teamBMember1Id
);

chatService.createGroupDateChatRoom(
        "과팅 채팅방",
        leaderMemberIds,
        memberIds
);

예시 3대3:

List<Long> leaderMemberIds = List.of(
        teamALeaderId,
        teamBLeaderId
);

List<Long> memberIds = List.of(
        teamAMember1Id,
        teamAMember2Id,
        teamBMember1Id,
        teamBMember2Id
);

chatService.createGroupDateChatRoom(
        "과팅 채팅방",
        leaderMemberIds,
        memberIds
);

예시 4대4:

List<Long> leaderMemberIds = List.of(
        teamALeaderId,
        teamBLeaderId
);

List<Long> memberIds = List.of(
        teamAMember1Id,
        teamAMember2Id,
        teamAMember3Id,
        teamBMember1Id,
        teamBMember2Id,
        teamBMember3Id
);

chatService.createGroupDateChatRoom(
        "과팅 채팅방",
        leaderMemberIds,
        memberIds
);

생성 결과:

room_type = GROUP
matching_mode = GROUP_DATE
leaderMemberIds에 있는 회원 → LEADER
memberIds에 있는 회원 → MEMBER
배열이면 이렇게 변환
Long[] memberIdArray = {
        teamAMember1Id,
        teamAMember2Id,
        teamBMember1Id,
        teamBMember2Id
};

List<Long> memberIds = Arrays.asList(memberIdArray);

chatService.createGroupDateChatRoom(
        "과팅 채팅방",
        leaderMemberIds,
        memberIds
);
